# OCTec
OCTec - A python based, automatic file handler and OCR engine

This is supposed to be a Pycharm Project, but you can use as you see fit, using your .idea in the project root.

The resource folder has to have the dependencies of the binaries: Tesseract-OCR, Poppler and gs10.04.0.

Run the pyinstaller on the .spec, and the Inno compiler on the .iss.

The license folder should have the necessary files to run the pyinstaller on the .spec, and the .iss needs the Inno compiler to create the installation wizard.
